# **App Name**: EcoLogic AI

## Core Features:

- Multi-Modal Consumption Auditor: An AI-powered document processor that functions as a tool to automatically extract and categorize data from uploaded utility bills, fuel receipts, and grocery lists to calculate exact emissions.
- Gemini Carbon Copilot: A sophisticated AI conversational tool that understands historical data to explain the 'why' behind emission spikes and provides personalized, context-aware coaching.
- Predictive Carbon Twin: A digital simulation engine that acts as a tool to forecast future environmental impact and score changes based on 'what-if' scenarios like switching to an EV or changing diets.
- Dynamic Eco Score & Explanation: A real-time sustainability index stored in Firestore that uses AI to provide transparent reasoning for score fluctuations based on recent behaviors and external factors.
- Green ROI Consultant: An AI advisory tool that calculates the precise financial and carbon payback periods for sustainability investments like solar panels or high-efficiency appliances.
- Hyper-Local Impact Challenges: Time-sensitive sustainability missions generated by AI that leverage local weather data and community trends to incentivize immediate emission reductions.
- Sustainability Milestone Vault: A robust record-keeping system utilizing Firestore to track historical carbon savings and manage user-earned achievements for long-term engagement.

## Style Guidelines:

- Primary color: Emerald Cyan (#2CDAB1) - a sophisticated, tech-forward green that balances nature with precision.
- Background color: Deep Carbon Charcoal (#121615) - a high-contrast dark scheme that emphasizes a high-end digital agency aesthetic.
- Accent color: Cobalt Mist (#2C80DA) - used sparingly for tooltips and actionable insights to provide contrast against the green primary.
- Headline font: 'Space Grotesk' (Sans-serif) - providing a machined, scientific feel appropriate for an AI auditor. Body font: 'Inter' (Sans-serif) - for objective clarity and maximum accessibility in long-form reports.
- Abstract, minimalist line icons with variable weights, reflecting data nodes and organic forms.
- A modular grid-based layout inspired by financial dashboards, prioritizing data density and clarity with expansive negative space to avoid cognitive overload.
- Progressive data-loading micro-animations that mimic scientific analysis sequences when the AI auditor is processing documents.
